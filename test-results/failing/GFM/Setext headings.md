# GFM - Setext headings

## [Example 50](https://spec.commonmark.org/0.29/#example-50)

This markdown:

````````````markdown
Foo *bar*
=========

Foo *bar*
---------

````````````

Should give output:

````````````html
<h1>Foo<em>bar</em></h1><h2>Foo<em>bar</em></h2>
````````````

But instead was:

````````````html
<p>Foo<em>bar</em>=========</p><p>Foo<em>bar</em></p><hr>
````````````
